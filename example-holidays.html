Date Range Picker with Holidays Feature

Examples demonstrating different ways to integrate holidays into the date range picker.

1. Simple String Format

The simplest way to add holidays using date strings in YYYY-MM-DD format:

$('#example1').daterangepicker({
    startDate: moment('2025-01-01'),
    endDate: moment('2025-01-15'),
    holidays: [
        '2025-01-01',
        '2025-01-06',
        '2025-01-15'
    ]
});

2. Object Format with Holiday Names

Add holiday names that appear as tooltips when hovering over dates:

$('#example2').daterangepicker({
    startDate: moment(),
    endDate: moment().add(30, 'days'),
    holidays: [
        { date: '2025-01-01', name: 'New Year\'s Day' },
        { date: '2025-02-14', name: 'Valentine\'s Day' },
        { date: '2025-07-04', name: 'Independence Day' }
    ]
});

3. Custom Styling with CSS Classes

4. Loading from External Source

Simulate loading holidays from an API or external file:

// Simulate API call
fetch('/api/holidays')
    .then(response => response.json())
    .then(holidays => {
        $('#example4').daterangepicker({
            holidays: holidays
        });
    });
